# EmotionMusicTherapy
Emotion-Based Music Therapy Room is an AI-powered application that combines facial emotion recognition with an interactive Unity-based virtual environment to provide a personalized music therapy experience. The system captures a user's facial expression using a webcam, detects the dominant emotion through the FER (Facial Emotion Recognition) library in Python, and automatically plays emotion-specific music. Simultaneously, a Unity 3D environment reads the detected emotion and dynamically changes the room's wall colors while displaying an emotion analysis dashboard with confidence percentages and personalized messages. By integrating Python, OpenCV, FER, Pygame, and Unity, this project demonstrates the practical application of artificial intelligence, computer vision, and interactive visualization to promote emotional well-being through an immersive and adaptive therapy environment.
